# Website lokal starten

Die Website darf für das 3D-Modell nicht direkt per Doppelklick als `file:///.../index.html` geöffnet werden.

Stattdessen `website-starten.cmd` doppelt anklicken. Die Website öffnet sich anschließend unter:

`http://localhost:8000/index.html`

Das Konsolenfenster muss während der Verwendung geöffnet bleiben. Mit `Strg + C` kann der lokale Server beendet werden.

Auf GitHub Pages oder einem normalen Webserver ist dieser lokale Starter nicht erforderlich.
